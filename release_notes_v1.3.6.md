# v1.3.6 - Eggie DocuFlow

本版本重点改进 PDF 批量转换、大任务进度提示、批量改名安全性，并在文档处理中增加可选的云 OCR 能力。

## 更新内容

- PDF 转图片可同时添加多个 PDF，也可选择文件夹自动添加其中的 PDF。
- 图片导出默认提升到高清 300 DPI，可选择 450 DPI 超清。
- 单页 PDF 直接生成图片；多页 PDF 按原文件名建立文件夹，并使用 `_1`、`_2` 顺序命名。
- 图片、PDF 页面和批量改名显示当前处理数量，并增加大批量数量保护。
- 大文件和多文件任务增加全局进度提示，避免用户误以为软件卡死。
- 图片和 PDF 页面预览采用缓存，勾选、排序和旋转时不再重复打开原文件。
- 批量改名减少连续输入引发的重复刷新，并阻止用过期预览结果执行改名。
- 文档处理可选使用百度智能云或阿里云 OCR，两个平台的密钥可分别保存并随时切换。
- OCR 文字、JSON 和日志会在全部生成成功后再一起保留，失败时不留下半成品。
- 调整主页、PDF 工具箱和页面整理按钮布局。

## 影响范围

- 影响 PDF 工具箱、批量改名、文档处理、设置页和大任务进度提示。
- 不改变 Excel 合并、Excel 拆分和 PDF 发票解析的原有业务规则。
- V1.3.5 发布和安装包保持不变。

## 适用范围

- macOS 11.0 及以上。
