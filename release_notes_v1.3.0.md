# v1.3.0 - Document Intelligence System

本次版本属于架构升级版本和产品形态升级版本，标志着 EggieExcelTool 从单一 Excel/PDF 工具升级为具备文档智能路由能力的稳定发布候选版本。

## 更新内容

- 文档智能路由系统上线
- 支持发票 / 合同 / 表格自动分类
- App入口完成（文件选择 + 一键处理）
- 模块化架构完成（core/parsers/exporters/utils）
- 真实PDF验证通过
- CLI增强与日志系统优化

## 发布说明

- 发票 PDF 输出结构化 Excel
- 合同 PDF 输出 Word 文档
- 表格 PDF 输出标准 Excel
- 扫描或无法识别 PDF 输出 UNKNOWN 文本
- 当前版本不引入 OCR
