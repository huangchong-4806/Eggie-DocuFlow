Excel Merge Tool

一个用于批量合并 Excel 文件的桌面工具，适合日常办公、财务对账、订单汇总、月报整理等场景。

功能介绍

本工具可以将多个 Excel 文件按照文件名顺序合并到一个 Excel 文件中，减少手动复制粘贴的工作量，提高表格整理效率。

主要特点

* 支持批量选择多个 Excel 文件
* 按文件名顺序依次合并
* 保留原始单元格格式
* 支持输出为新的 Excel 文件
* 操作简单，适合非技术人员使用
* 适用于日常办公、数据汇总、财务对账等场景

使用场景

这个工具适合处理类似下面的情况：

* 每月需要合并多份 Excel 表格
* 多个部门提交的表格需要汇总
* 订单明细、付款明细、发票明细需要整理到一起
* 不想反复手动复制、粘贴、调整格式

使用方法

1. 打开 Excel 合并工具
2. 选择需要合并的 Excel 文件
3. 确认文件顺序
4. 选择输出位置
5. 点击开始合并
6. 等待生成新的合并文件

注意事项

* 建议提前确认文件名顺序，因为工具会按照文件名顺序进行合并
* 建议被合并的 Excel 文件表头结构保持一致
* 如果原文件格式较复杂，合并后建议人工复核一次
* 使用前建议备份原始文件，避免误操作

适用平台

目前主要用于本地电脑环境。

后续可根据需要支持：

* macOS 桌面版
* Windows 桌面版
* 更多 Excel 格式兼容
* 自定义合并规则

项目说明

这个项目主要是为了解决日常办公中频繁合并 Excel 文件的问题，让重复性表格整理工作变得更简单。
