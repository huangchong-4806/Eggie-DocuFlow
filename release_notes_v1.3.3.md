# v1.3.3 - Eggie DocuFlow

本版本新增“PDF 工具箱”，并增强批量 PDF 文档处理速度与正式合同增强排版效果。

## 更新内容

- 新增 PDF 工具箱入口。
- 页面整理支持缩略图预览和拖拽排序。
- 支持多选页面后左转、右转、旋转 180 度、删除和拆分导出。
- 多 PDF 合并支持默认文件名，并允许用户手动修改输出文件名。
- 新增 PDF 安全压缩，支持三档压缩，并显示预计压缩后大小和预计缩小比例。
- 新增多张图片合成 PDF，支持缩略图预览、拖拽排序、双击放大和文件夹导入。
- 图片转 PDF 会自动过滤非图片和无法读取的文件。
- 新增 PDF 每页导出 JPG 或 PNG 图片，默认导出 JPG。
- 添加文件和文件夹会记住上次打开位置。
- 优化缩略图卡片布局，避免图片和文字重叠。
- 所有 PDF 工具操作都会生成 txt 日志，记录来源文件、处理动作、页码、输出位置和失败原因。
- 批量 PDF 处理默认同时处理 2 个文件。
- 批量处理日志会记录每个文件的开始、成功、失败原因和输出位置，便于排查问题。
- 并发输出时会再次确认结果文件名，避免多个任务生成同名文件时互相覆盖。
- 优化正式合同增强排版，改善跨页文字合并、段落清理和表格续接效果。

## 影响范围

- 影响文档智能处理中的批量 PDF 处理。
- 影响正式合同增强排版输出。
- 不改变 Excel 合并、Excel 拆分、批量改名和单个发票转换的原有操作方式。
