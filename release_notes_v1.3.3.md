# v1.3.3 - Eggie DocuFlow

本版本新增“PDF 工具箱”，重点覆盖 PDF 页面整理、PDF 压缩、图片转 PDF 和 PDF 转图片。

## 更新内容

- 新增 PDF 工具箱入口。
- 页面整理支持缩略图预览和拖拽排序。
- 支持多选页面后左转、右转、旋转 180 度、删除和拆分导出。
- 多 PDF 合并支持默认文件名，并允许用户手动修改输出文件名。
- 新增 PDF 安全压缩，支持三档压缩，并显示预计压缩后大小和预计缩小比例。
- 新增多张图片合成 PDF，支持缩略图预览、拖拽排序、双击放大和文件夹导入。
- 图片转 PDF 会自动过滤非图片和无法读取的文件。
- 新增 PDF 每页导出 JPG 或 PNG 图片，默认导出 JPG。
- 添加文件和文件夹会记住上次打开位置。
- 优化缩略图卡片布局，避免图片和文字重叠。
- 所有 PDF 工具操作都会生成 txt 日志，记录来源文件、处理动作、页码、输出位置和失败原因。
