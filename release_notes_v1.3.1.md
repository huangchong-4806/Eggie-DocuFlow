# v1.3.1 - Eggie DocuFlow

本版本是在 v1.3.0 文档智能路由基础上的小版本增强，重点是让合同和表格 PDF 的输出更接近日常办公可编辑格式。

## 新增功能

- 文档智能处理页面新增“增强排版转换（适合合同和表格）”选项。
- 用户仍然只需要选择 PDF，系统自动判断发票、合同、表格或未知文档。
- 合同增强模式会套用正式合同样式，统一字体、字号、页边距、行距和标题格式。
- 合同附件中的表格会尽量还原为可编辑 Word 表格。
- 表格增强模式会尽量保留边框、列宽、行高和打印方向。
- 软件界面新增当前版本号显示，后续随 `version.py` 自动更新。

## 稳定性说明

- 未修改 v1 核心 router。
- 未修改现有 parser 主流程。
- 发票和未知文档不会强行套用合同模板。
- 扫描件仍不启用 OCR，会输出 UNKNOWN 文本说明。
