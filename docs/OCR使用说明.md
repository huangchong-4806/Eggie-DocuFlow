# Eggie DocuFlow OCR 使用说明

版本：1.0
更新日期：2026 年 7 月 14 日
适用软件：Eggie DocuFlow（Eggie文档处理系统）

## 一、这个功能是做什么的

OCR 用于读取扫描件、照片式 PDF 中的文字。它已融合在软件的“文档处理”页面中，不是一个单独的软件或菜单。

- 不勾选“扫描页使用云 OCR”时，原有文档处理方式完全不变。
- 勾选后，软件先在本机读取可复制的文字页，只把扫描图片页发送给您选择的 OCR 平台。
- OCR 文字会继续进入同一个文档分类和输出流程。
- 如果只想要文字，可在同一页面点击“仅提取文字”。
- 原 PDF 只读取，不会被修改或覆盖。

## 二、使用前必须了解的关系

1. 您自行到百度智能云或阿里云注册、实名、开通 OCR，并同意该平台的服务条款、隐私规则和计费规则。
2. OCR 服务合同、账号、额度和费用发生在您与所选 OCR 平台之间。
3. Eggie DocuFlow 是运行在您电脑上的本地工具，不是百度或阿里云的代理商、转售商或 OCR 服务提供方。
4. 软件设计上不将您的密钥或文档发送给 Eggie DocuFlow 开发者，也没有开发者中转服务器。
5. 云 OCR 不是“完全本地处理”。扫描页图片会离开您的电脑，由所选平台按其规则处理。

## 三、文档和密钥会去哪里

### 1. 可复制文字的 PDF 页

只在您的 Mac 本机读取，不会发送给 OCR 平台。

### 2. 扫描图片页

软件会先显示页数和平台名称，等您确认后，再通过加密网络直接发送给所选平台。软件发送的是当前扫描页生成的图片，不使用公开图片网址，不通过开发者服务器中转。

阿里云官方 OCR 安全说明当前表示，公共云 OCR 识别是无状态的，原始图片不保留并在结果返回后释放。该说明可能随平台政策调整，请以使用当日的官方页面为准。

### 3. API Key / AccessKey

密钥保存在当前 Mac 用户的个人配置区，文件名为 `.env`，仅当前用户可读写。密钥不写入：

- 软件项目和安装包；
- 处理结果；
- 处理日志；
- PDF 或 Excel 文件。

请不要将 `.env` 文件、截图或密钥文字发给他人。百度智能云官方文档明确提醒，密钥泄露可能导致资源被盗用，并提醒用户警惕不可信或非法的第三方插件。建议为本软件单独创建低权限密钥，设置费用或用量提醒，不与其他业务共用。

### 4. 处理日志

日志只记录排查问题所需的信息：文件路径、页数、本机处理页数、云 OCR 页数、处理时间、平台返回的请求编号、错误码和结果文件路径。日志不记录密钥，也不复制文档正文。

## 四、准备百度智能云 OCR

1. 打开百度智能云 OCR 官方新手指南。
2. 按平台提示注册、实名并开通文字识别服务。
3. 在平台中创建一个专门用于 Eggie DocuFlow 的 OCR 应用。
4. 获取该应用的 `API Key` 和 `Secret Key`。
5. 查看当前免费额度、付费规则和用量告警设置。
6. 不要使用有其他重要业务权限的共用密钥。

官方新手指南：https://cloud.baidu.com/doc/OCR/s/dk3iqnq51
官方接口说明：https://cloud.baidu.com/doc/OCR/s/vk3h7y58v
官方免费额度说明：https://cloud.baidu.com/doc/OCR/s/fk3h7xu7h

## 五、准备阿里云 OCR

1. 打开阿里云 OCR 官方快速使用说明。
2. 按平台提示注册、实名并开通 OCR 服务。
3. 不建议使用主账号密钥。请创建专用的 RAM 子用户，只授予调用 OCR 所必需的权限。
4. 为该 RAM 用户创建 `AccessKey ID` 和 `AccessKey Secret`。`AccessKey Secret` 通常只在创建时显示一次，请妥善保管。
5. 查看当前免费额度、按量付费规则和用量告警设置。

## 六、在 Eggie DocuFlow 中配置

1. 打开软件，点击左下角“设置”。
2. 在“第三方服务”中选择“百度智能云 OCR”或“阿里云 OCR”，它会作为文档处理和未来 PDF 比对共用的默认平台。
3. 填写对应平台的两项密钥，点击“保存当前平台密钥”。
4. 如需测试，可点击“保存并测试连接”。百度连接测试只验证密钥，不上传文档页面；阿里云连接测试会发送一张由软件生成的 TEST 图片，可能占用 1 次调用额度，不包含您的文档。
5. 点击“完成”返回软件。进入“文档处理”后，可以直接看到当前平台和是否已配置。
6. 连接测试只能证明当时的密钥和网络可用，不代表后续不会受到额度、费用、平台维护或网络影响。

## 七、处理一份 PDF

### 方式 A：识别后继续文档处理

1. 在“待处理 PDF”中选择文件。
2. 确认结果保存文件夹。
3. 勾选“扫描页使用云 OCR”。
4. 点击原有的“一键识别并处理”。
5. 软件会先检查哪些页面是扫描页。如果需要发送，会显示平台名称、扫描页数和页码。
6. 确认您有权处理后，再点击继续。
7. OCR 文字会继续进入原有文档分类和输出流程。

说明：扫描件的版式、表格和发票字段可能因图片清晰度、倾斜、遮挡或 OCR 误差而无法可靠结构化。为避免生成误导数据，当软件不能可靠识别行列或发票明细时，会改为输出 OCR 文字，而不会强行生成不完整的 Excel。

### 方式 B：只提取文字

1. 选择 PDF 和保存文件夹。
2. 在“扫描件文字识别”区域选择平台。
3. 点击“仅提取文字”。
4. 如果 PDF 只有可复制文字，不需要密钥，也不会调用云 OCR。
5. 如果有扫描页，按弹窗提示确认后再发送。

## 八、会生成哪些文件

点击“仅提取文字”后，会生成三个不覆盖原文件的结果：

1. `原文件名_文字提取.txt`：日常查看和复制文字用。
2. `原文件名_文字提取.json`：保留页码、文字位置、识别可信度和请求编号，为以后 PDF 比对保留统一数据。普通用户无需手动修改。
3. `原文件名_文字提取_日志.txt`：记录页数、处理方式、请求编号和文件生成状态，便于排查。

## 九、额度、费用与政策变化

- 每个扫描页通常会产生 1 次 OCR 调用。文本页不调用云 OCR。
- 截至 2026 年 7 月 14 日，百度智能云和阿里云都有官方免费额度或试用规则，但不同接口的额度不同。
- 免费额度、价格、接口名称、实名要求、数据处理规则和服务可用性都可能变化。
- 软件只显示预计调用页数，不能代表平台的最终账单。实际额度和费用以您的云平台控制台为准。
- 建议在云平台设置费用中心告警、用量告警或限额，并定期检查调用记录。

1. 只处理您自己拥有、受托处理或已取得必要授权的文档。
2. 如文档包含他人的个人信息、敏感个人信息、未公开商业资料、客户资料或内部文件，请在上传前完成必要的授权、告知、同意、审批或脱敏。
3. 不要上传国家秘密、法律禁止传输的数据、来源不合法的资料，或未经单位允许使用公共云处理的敏感文档。

6. 您需要自行承担所选平台的账号管理、权限、额度、费用、服务中断和条款变化带来的影响。

## 十一、删除或更换密钥

### 只删除这台 Mac 上保存的密钥

1. 点击软件左下角“设置”。
2. 在“第三方服务”中选择对应平台。
3. 点击“删除本机密钥”。

这只会删除当前 Mac 上的保存内容，不会删除云平台中的账号或 AccessKey。

### 密钥可能泄露时

1. 立即登录对应云平台，停用或删除旧密钥。
2. 查看平台调用记录和账单，确认是否有异常用量。
3. 创建权限更小的新密钥，然后在软件中更换。
4. 如有异常扣费或数据风险，立即联系平台客服和所在单位的安全负责人。

## 十二、常见问题

### 提示“密钥未配置”

请点击软件左下角“设置”，在“第三方服务”中填写并保存当前平台的两项密钥。

### 提示“密钥验证失败”

检查是否复制了多余空格，密钥是否已停用，账号是否已开通 OCR，子用户是否拥有 OCR 调用权限。

### 提示额度不足或调用过快

登录所选平台查看剩余额度、账单和并发限制。软件对临时网络或限流错误最多自动重试 1 次，不会无限重试或无限产生费用。

### 识别结果有错字、丢字或顺序不对

先检查原文件是否清晰、倾斜、遮挡或存在手写内容。重要结果必须人工对照原 PDF 修正，不要直接用于付款、税务或合同决策。

### 处理失败会不会损坏原 PDF

不会。软件只读取原 PDF，结果另存为新文件。处理失败时不会用不完整文件覆盖原文件。
