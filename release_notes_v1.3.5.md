# v1.3.5 - Eggie DocuFlow

本版本把已确认的朴素科技风预览落实到完整软件，并保留现有全部处理能力。

## 更新内容

- 左侧菜单在工作台和全部功能页面持续显示，当前页面使用蓝色高亮。
- 主界面采用白色背景、浅灰卡片和科技蓝强调色。
- 删除首页重复的“打开 Excel 合并”按钮和无真实数据的信息面板。
- 统一 Excel 合并、Excel 拆分、发票解析、文档处理、批量改名和 PDF 工具箱的标题布局。
- 放大并加粗全部数字输入框的上下箭头和点击区域。
- 发票结果文件改为使用发票号码命名，重名时自动添加序号。
- 发票汇总日志补充销售方名称和销售方税号。
- 修复包含真实内容流的 PDF 无法完成压缩的问题。
- 统一软件中英文名称、版本号和 macOS 显示名称。

## 影响范围

- 影响主界面、全部工具页、设置入口和 macOS 打包信息。
- 影响 PDF 发票结果文件名和发票台账日志字段。
- 提升 PDF 压缩对真实文件的兼容性。
- 不改变 Excel、文档处理、批量改名和 PDF 工具箱的原有处理规则。
